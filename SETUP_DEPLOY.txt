# 🚀 Deploy Rápido (5 minutos)

## **OPÇÃO MAIS FÁCIL: Render.com**

### Passo 1: Criar Repositório GitHub
```bash
cd c:\Users\Professor\sistema_escolar
git init
git add .
git commit -m "Sistema de Agendamento Escolar"
```

Depois crie um repositório vazio em https://github.com/new e execute:
```bash
git remote add origin https://github.com/SEU_USUARIO/sistema_escolar.git
git branch -M main
git push -u origin main
```

### Passo 2: Deploy no Render
1. Acesse https://render.com
2. Clique em "Sign Up" (use GitHub para mais fácil)
3. Clique em "New +" → "Web Service"
4. Conecte seu repositório `sistema_escolar`
5. Configure assim:
   - **Name:** `sistema-escolar` (ou difícil em outro projeto)
   - **environment:** Python 3
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `gunicorn wsgi:app`
6. Clique "Create Web Service"
7. **Pronto! Aguarde 2-3 minutos**

Você receberá uma URL tipo: `https://sistema-escolar-xxxx.onrender.com` 🎉
