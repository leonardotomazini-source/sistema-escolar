✅ PASSO 1: ABRA GITHUB DESKTOP

✅ PASSO 2: SELECIONE SEU REPOSITÓRIO
   - No lado esquerdo você vê "Current Repository"
   - Clique no dropdown
   - Selecione "sistema-escolar"

✅ PASSO 3: VEJA AS MUDANÇAS
   - Na aba "Changes" você verá:
     • app.py (modificado)
     • Alguns arquivos novos
   
✅ PASSO 4: DESCREVA O COMMIT
   Na caixa "Summary" escreva:
   
   "Fix: caminhos absolutos no database e melhorias"

✅ PASSO 5: CONFIRME
   Clique em "Commit to main"

✅ PASSO 6: ENVIE
   Clique em "Push origin"
   
   Deve aparecer uma mensagem: "Successfully pushed to origin"

✅ PASSO 7: PRONTO!
   Render vai detectar e fazer redeploy em ~2-3 minutos
